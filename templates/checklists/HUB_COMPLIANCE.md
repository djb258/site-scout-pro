# Hub Compliance Checklist (Required)

This checklist must be completed before any hub can ship.
No exceptions. No partial compliance.

---

## Pre-Ship Checklist

- [ ] Hub purpose and boundary defined
- [ ] Hub PRD exists and is versioned
- [ ] Spokes (capabilities) explicitly listed
- [ ] Connectors (API / CSV / Event) defined
- [ ] Approved tools listed at hub level
- [ ] No sideways hub-to-hub calls
- [ ] Guard rails implemented (schema, validation, permissions)
- [ ] Kill switch defined and testable
- [ ] Promotion gates defined
- [ ] Logging & observability implemented
- [ ] Failure modes documented
- [ ] Human override rules defined

---

## Compliance Rule

If any box is unchecked, this hub may not ship.
